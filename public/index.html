Gadget-WikiForum provide a front-end structured discussion page with JavaScript for MediaWiki app.

Quick Start

This is a quick-start guide. It allows you to load a forum with basic functions using a preset scheme.

Step 1: Create a gadget page in your wiki

For example:

MediaWiki:Gadget-WikiForum.js

Step 2: Invoke Core

mw.loader.load('https://github.com/Fandom-zh/Gadget-WikiForum/dist/core.min.js')

Step 3: Invoke Default Theme

mw.loader.load('https://github.com/Fandom-zh/Gadget-WikiForum/dist/theme/default.min.js')

Step 4: Invoke Default Loader and Configure

mw.loader.load('https://github.com/Fandom-zh/Gadget-WikiForum/dist/loader/default.min.js')
window.WikiForumNS = [/* Your Forum NamespaceNumber here */]

Step 5: Enable the gadget

Insert a new line to MediaWiki:Gadgets-definition to regist the gadget. 

* WikiForum[ResourceLoader|hidden|default]|WikiForum.js
